## Getting Started

1. **Clone the repository**
```bash
git clone https://github.com/yourusername/cit-class-routine-manager.git
cd cit-class-routine-manager
```

2. **Install dependencies**
```bash
npm install
```

3. **Start development server**
```bash
npm run dev
```

4. **Open browser**
Visit `http://localhost:3000`
